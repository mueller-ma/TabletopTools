Este aplicativo disponibiliza diversas ferramentas uteis para jogos de tabuleiro:

* Dados (D3, D4, D6, D8, D10, D12, D20, D100)
* [Rot13](https://es.wikipedia.org/wiki/ROT13)
* Conversor numérico: Decimal <=> Binário <=> Hexadecimal
* Timer de 5 minutos
* Campainha para dos jogadores

Mais ferramentas estão sendo planejadas.
